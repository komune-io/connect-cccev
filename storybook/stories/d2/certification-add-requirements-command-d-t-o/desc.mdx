


### Command

Type: `CertificationAddRequirementsCommandDTO`

---

***id*** `String`

Id of the certification to add the requirements to.



---

***parentId*** `String?`

Optional RequirementCertification id in which to add the requirements. If null, the requirements will be added to the root of the certification itself.



---

***requirementIdentifiers*** `List<String>`

Identifiers of the requirements that the certifications will have to fulfill.
