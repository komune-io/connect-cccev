


### Event

Type: `CertificationAddedRequirementsEventDTO`

---

***id*** `String`

Id of the certification the requirements have been added to.



---

***parentId*** `String?`

Optional RequirementCertification id in which the requirements have been added. If null, the requirements have be added to the root of the certification itself.



---

***requirementCertificationIds*** `List<String>`

List of the added requirement certifications.
